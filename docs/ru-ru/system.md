# Системные настройки

> Введение

![System](/_images/setting/setting_1.png) 

В `System` администратор изменяет общие параметры IPTV: импорт лицензии, бэкап БД, настройки безопасности внешних API.

## Setting

> Введение

![Setting - Setting ](/_images/setting/setting_2.png) 

В `Setting` задаются данные, синхронизируемые на терминалы: фоновая графика, формат времени и прочие параметры.

<font color="red">**Monetary Unit**</font>: локальный символ валюты (отображается в Shopping).

<font color="red">**Paid Days**</font>: срок доступа при покупке пакета/фильма.

<font color="red">**Consumption Mode**</font>: способ оплаты гостя. `Prepaid` — расходуются средства счёта; `Advance` — расход в кредит (долг на номер).

<font color="red">**Home Page Show**</font>: фон главной — Image (картинка) или Video (видео).

<font color="red">**Time Format**</font>: формат времени на терминале.

<font color="red">**Favorite Operation**</font>: очищать ли избранное при выезде.

<font color="red">**Request ring**</font>: загрузка тона оповещений (устарело).

<font color="red">**Welcome Background**</font>: фоновое изображение приветствия (синхронизируется на ТВ).

<font color="red">**Home Background**</font>: фон главной страницы.

<font color="red">**Secondary Menu Background**</font>: фон второго уровня меню.

<font color="red">**LOGO**</font>: логотип для ТВ и мобильных клиентов.

<font color="red">**Live Player Watermark**</font>: устарело.

<font color="red">**Vod Player Watermark**</font>: устарело.

<font color="red">**City**</font>: город для получения прогноза погоды.

<font color="red">**Enable Remote Assistance**</font>: устарело.

## Version

> Введение

![APK Version Management](/_images/setting/setting_3.png) 

В APK Version Management настраиваются политики обновления (обязательные/необязательные) для разных терминалов.

`APK Upgrade` — загрузить APK; после выбора файла система считает версию и покажет её в списке, администратор проверяет корректность.
