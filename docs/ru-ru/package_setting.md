# Настройка подписок (Packages)

> Введение

![Login Page](_images/11.png)

В Package Information создаются подписки для продажи: имя, цена, срок действия и привязка Live/VOD. Подписки можно редактировать и удалять.

Нажмите `Add`, чтобы создать подписку.

![Login Page](_images/11-2.png)

1. <font color="red">**Package Name**</font>: отображаемое название пакета.

2. <font color="red">**Price**</font>: цена пакета.

3. <font color="red">**Expiration Date**</font>: число срока действия.

4. <font color="red">**Packages Type**</font>: единица срока (`Day` / `Month` / `Year`); комбинируется с Expiration Date (например 1 + Month = 1 месяц).

5. <font color="red">**Live Program Select**</font>: live-программы, включённые в пакет.

6. <font color="red">**VOD Program Select**</font>: фильмы (VOD), включённые в пакет.
