# Сервис отеля

>Введение

![Сервис отеля](_images/hotel/hotel_1.png) 

Модуль `Hotel Service` позволяет администраторам настраивать категории и элементы услуг для системы IPTV в номере. Администраторы могут просматривать, принимать, завершать или отменять заказы гостей в режиме реального времени. Гости могут размещать заказы услуг непосредственно со своего телевизора, в то время как администраторы управляют всем процессом заказа через платформу.

## Тип сервиса отеля

>Введение

![Тип сервиса отеля](_images/hotel/hotel_2.png) 

В разделе `Hotel Service Type` администратор создает, изменяет и удаляет категории сервиса отеля.

Нажмите кнопку `Add`, чтобы создать новую категорию сервиса отеля

![Тип сервиса отеля](_images/hotel/hotel_3.png)

<font color="red">**ID**</font>: В `ID` система IPTV автоматически создает номер ID, который используется для сортировки различных категорий; чем меньше число, тем выше его позиция в отображении

<font color="red">**Name**</font>: `Name` используется для отображения сервиса отеля категории.

## Список сервисов отеля

>Введение

![Сервис отеля](_images/hotel/hotel_5.png)

![Сервис отеля](_images/hotel/hotel_4.png)

На странице `Hotel Service List` администратор управляет элементами сервиса отеля, доступными для заказа в номерах, путем создания, редактирования или удаления продуктов по мере необходимости.

Нажмите кнопку `Add`, чтобы создать новый сервис для онлайн-заказа.

![Список сервисов отеля](_images/hotel/hotel_6.png)

<font color="red">**Image**</font>: В `Image` загрузите изображения сервиса отеля в Изображение, и изображения будут отображаться на терминале; в настоящее время поддерживается загрузка только одного изображения.

<font color="red">**Background picture**</font>: Фоновое изображение, которое будет отображаться на терминале, когда пользователь выбирает или нажимает на элемент сервиса отеля.

<font color="red">**Name**</font>: `Name` используется для идентификации названия сервиса отеля, отображаемого на терминале.
<font color="red">**Price**</font>: В `Price` используется для отображения цены за единицу, по которой продается сервис отеля.

<font color="red">**Hotel Service Type**</font>: Выберите `Hotel Service Type` для услуги, и услуга появится в соответствующем списке категорий на терминале.

<font color="red">**Description**</font>: В `Description` используется для описания конкретной информации и деталей об этом сервисе отеля.
## Состояние заказа

>Введение

![Состояние заказа](_images/hotel/hotel_7.png ':size=40%') 

На этой странице администратор может просматривать детали заказа элементов, отправленного подписчиком с терминала. Когда администратор получает заказ подписчика, он может `receive` заказ с помощью кнопки RECEIVE или `cancel` заказ подписчика с помощью кнопки DELETE. После принятия заказа подписчика, когда заказ будет выполнен, администратор может снова нажать кнопку Finish, чтобы `complete` весь процесс заказа.
