# وثائق Hooray IPTV عبر الإنترنت

> بناءً على Hooray Hotel IPTV Platform v1.1.3_Release_Date_2025-12-11.

## نظرة عامة

يوثق هذا الموقع كيفية تكوين وتشغيل منصة إدارة Hooray Hotel IPTV. يوفر إرشادات خطوة بخطوة للتثبيت والتكوين والعمليات اليومية للنظام.

## نظرة سريعة على الميزات

- **إدارة الأجهزة والعملاء:** تسجيل وإدارة أجهزة الاستقبال والعملاء، وتجميع الأجهزة، ودفع التكوينات وتحديثات OTA عبر وحدة التحكم في الإدارة.
- **التلفزيون المباشر:** تكوين القنوات الحية والمصادر ودليل البرامج الإلكتروني (EPG)، وإدارة حزم القنوات والجدولة الزمنية.
- **الفيديو حسب الطلب (VOD):** تحميل وتصنيف وإدارة مكتبات المحتوى حسب الطلب مع عناصر التحكم في التشغيل والبيانات الوصفية.
- **الإعلانات والعروض الترويجية:** جدولة وإدارة إعلانات الغرف وإعلانات القنوات والمحتوى الترويجي مع دعم إدارة الوسائط.
- **الحزم والإعدادات:** إنشاء حزم القنوات/الحزم، وتكوين قواعد الوصول، وإدارة الإعدادات المتعلقة بالحزم.
- **خدمات وتكاملات الفنادق:** توفير ميزات تتمحور حول الفنادق مثل معلومات المرافق ومعلومات الرحلات والطقس والتسوق ورسائل الضيوف.
- **الصلاحيات والأمان:** التحكم في الوصول القائم على الأدوار، وصلاحيات المشغلين، وإدارة السلطة للعمليات الآمنة.
- **إدارة النظام:** إعدادات النظام والنسخ الاحتياطي والتسجيل وأدوات الصيانة للتشغيل المستقر واستكشاف الأخطاء.
- **التوطين والأصول:** الوثائق والأصول المحلية متاحة ضمن `docs/` و `docs/ru-ru/`، مع الصور في `docs/_images/`.

## البدء

- اقرأ `quickstart.md` للحصول على إرشادات الإعداد الأولي.
- قم بتكوين بيانات البت والقنوات باستخدام `live-setting.md` و `vod_setting.md`.
- انشر وأدر APKs العملاء باستخدام `apk.md` وراقب الأجهزة عبر `client_manage.md`.

## أين تجد المساعدة

- ارجع إلى صفحات الموضوعات في `docs/` للحصول على أدلة وأمثلة مفصلة.
- استخدم `README.md` و `LICENSE` للحصول على معلومات على مستوى المشروع ومعلومات الترخيص.
