## Network Setting

>مقدمة

في `Network Setting`، يمكنك عرض وتكوين عنوان IP والمنفذ لخادم IPTV، بالإضافة إلى عنوان IP وعنوان MAC للاتصال المحلي.

![Network Setting ](_images/apk/apk_2.png) 

<font color="red">**Server IP**</font>: في `Server IP`، يمكن ملء كل من عناوين IP وأسماء النطاقات، ويجب أن تشير إلى خادم IPTV. إذا فشل عنوان IP أو اسم النطاق المملوء في الوصول إلى خادم IPTV، فيجب إعادة تكوين الجهاز بحيث يمكن للجهاز الوصول إلى IP أو اسم النطاق لخادم IPTV وبالتالي إجراء اتصال بيانات مع خادم IPTV.

<font color="red">**Server Port**</font>: في `Server Port`، فقط عن طريق ملء منفذ الخادم الصحيح يمكن إنشاء اتصال مع الخادم. منفذ الخادم هو نفسه رقم المنفذ على صفحة الويب. فقط املأ رقم المنفذ ذلك.

<font color="red">**Local Network**</font>: يشير `Local network` إلى عنوان IP الذي حصل عليه الجهاز الحالي، والذي يتم توفيره بواسطة نظام Android. يرجى التأكد من أن هذا العنوان يمكنه التواصل مع IP أو اسم النطاق الخاص بالخادم.

<font color="red">**Local MAC**</font>: يمثل `Local MAC` معلومات MAC التي حصل عليها الجهاز الحالي. سيتم تخزين معلومات MAC هذه في النظام كبيانات اعتماد فريدة للنظام. في نظام الإدارة، سيتم استخدام معلومات MAC لإدارة الجهاز المقابل وتعيين رقم غرفة الفندق.
