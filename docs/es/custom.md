# Configuración personalizada

Personalice la experiencia del usuario en la plataforma Hooray Hotel IPTV.

## Interfaz de usuario

### Tema y diseño

1. Vaya a **Configuración personalizada** → **Interfaz**
2. Seleccione tema:
   - Claro
   - Oscuro
   - Personalizado
3. Configure colores de marca
4. Cargue logotipos

### Idioma

- Seleccionar idioma predeterminado
- Configurar fechas y horarios
- Establecer moneda

## Contenido por defecto

### Canales destacados

1. Seleccione canales para mostrar primero
2. Configure orden de visualización
3. Establezca contenido destacado

### Inicio personalizado

- Configurar widget de inicio
- Agregar recomendaciones
- Personalizar menú principal

## Control parental

### Restricciones de contenido

1. Vaya a **Control parental**
2. Establezca clasificaciones permitidas
3. Configure PIN requerido
4. Defina horarios de visualización

### Bloqueo de contenido

- Bloquear categorías
- Restringir canales
- Aplicar filtros

## Preferencias de usuario

### Perfiles

1. Crear perfiles por habitación
2. Personalizar para cada perfil
3. Guardar preferencias
4. Sincronizar entre dispositivos

## Información de marca del hotel

### Detalles del hotel

- Nombre del hotel
- Logo
- Información de contacto
- Políticas

### Información personalizada

- Mensajes de bienvenida
- Información sobre servicios
- Promociones locales
- Enlaces a recursos

## Resolución de problemas

- Restablecer configuración predeterminada
- Verificar permisos
- Comprobar compatibilidad
