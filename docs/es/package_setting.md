# Configuración de paquetes

Gestione paquetes de canales y servicios en la plataforma Hooray Hotel IPTV.

## Crear paquetes

### Nuevo paquete

1. Vaya a **Paquetes** → **Crear paquete**
2. Nombre el paquete (ej: "Paquete básico")
3. Describa contenido
4. Seleccione canales incluidos
5. Configure acceso
6. Establezca precio (si es aplicable)
7. Guarde

### Tipos de paquetes

- Paquete básico
- Paquete premium
- Paquete deportivo
- Paquete películas
- Paquete personalizado

## Gestión de contenido

### Asignar canales

1. Seleccione paquete
2. Agregue canales
3. Configure orden
4. Establezca disponibilidad
5. Guarde

### Contenido VOD

- Agregar películas
- Agregar series
- Agregar documentales
- Configurar restricciones

## Configuración de acceso

### Permisos de usuario

1. Seleccione paquete
2. Defina quién puede acceder:
   - Todos los huéspedes
   - Huéspedes específicos
   - Habitaciones específicas
   - Período de tiempo
3. Guarde

### Restricciones

- Restricción de edad
- Filtro de contenido
- Restricción de horario
- Control por PIN

## Precios y facturación

### Modelo de precios

- Tarifa fija
- Por evento
- Por dispositivo
- Por período

### Período de disponibilidad

- Diario
- Semanal
- Mensual
- Por estancia

## Promociones

### Ofertas especiales

1. Crear promoción
2. Asignar paquete
3. Establecer:
   - Descuento
   - Período
   - Dispositivos objetivo
   - Límite de usuarios
4. Activar

### Pruebas gratuitas

- Período de prueba
- Limitaciones
- Conversión automática
- Recordatorio de vencimiento

## Reportes

### Análisis de paquetes

- Suscripciones activas
- Ingresos por paquete
- Utilización de contenido
- Retención de clientes

## Resolución de problemas

- Verificar asignación de paquete
- Comprobar permisos de acceso
- Revisar configuración de facturación
- Verificar disponibilidad de contenido
