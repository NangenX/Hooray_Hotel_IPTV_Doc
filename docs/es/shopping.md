# Compras

Gestione sistema de compras e integración comercial en la plataforma Hooray Hotel IPTV.

## Catálogo de productos

### Agregar productos

1. Vaya a **Compras** → **Catálogo**
2. Haga clic en **Nuevo producto**
3. Ingrese detalles:
   - Nombre
   - Descripción
   - Precio
   - Imagen
   - Categoría
   - Existencias
4. Guarde

### Categorías

- Alimentos y bebidas
- Servicios del hotel
- Artículos de regalo
- Entretenimiento
- Otros

### Gestión de inventario

- Rastrear existencias
- Configurar alertas
- Definir niveles mínimos
- Actualizar precios

## Pedidos

### Procesar pedidos

1. Cliente realiza pedido desde IPTV
2. Sistema registra pedido
3. Notificar personal del hotel
4. Procesar entrega/servicio
5. Confirmar finalización

### Estados de pedido

- Pendiente
- Confirmado
- En proceso
- Entregado
- Cancelado

## Pagos

### Métodos de pago

- Cargo a habitación
- Tarjeta de crédito
- Monedero digital
- Efectivo a la entrega

### Seguridad

- Cifrado de datos
- PCI compliance
- Protección de fraude
- Logs de transacción

## Reportes

### Análisis de ventas

1. Vaya a **Reportes**
2. Seleccione período
3. Analice:
   - Productos más vendidos
   - Ingresos
   - Clientes
   - Tendencias
4. Exporte datos

## Integración

### Sistemas externos

- Punto de venta (POS)
- Sistema de facturación
- Inventario central
- Pasarela de pago

## Resolución de problemas

- Verificar disponibilidad de productos
- Comprobar métodos de pago
- Revisar registros de transacciones
- Verificar sincronización de inventario
