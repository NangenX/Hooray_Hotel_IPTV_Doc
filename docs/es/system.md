# Configuración del sistema

Gestione la configuración básica del sistema de la plataforma Hooray Hotel IPTV.

## Configuración general

### Información del sistema

1. Vaya a **Configuración del sistema** → **General**
2. Configure:
   - Nombre del sistema
   - Zona horaria
   - Idioma predeterminado
   - Formato de fecha/hora

### Actualización del sistema

- Verificar actualizaciones disponibles
- Descargar actualizaciones
- Programar actualización automática
- Restaurar versión anterior

## Red

### Configuración de red

1. Vaya a **Red**
2. Configure:
   - Dirección IP
   - Gateway predeterminado
   - Servidores DNS
   - Conexión VPN (si es necesaria)

### Conectividad

- Prueba de velocidad
- Diagnóstico de conectividad
- Configurar proxy
- Firewall

## Almacenamiento

### Gestión de espacio

1. Ver uso de almacenamiento
2. Identificar archivos grandes
3. Limpiar archivos temporales
4. Configurar cuotas

### Copias de seguridad

1. Vaya a **Copias de seguridad**
2. Configure:
   - Horario de copia
   - Ubicación de almacenamiento
   - Retención de copias
3. Realice copia manual

### Restauración

- Restaurar desde copia
- Seleccionar punto de restauración
- Verificar integridad

## Registros

### Ver registros

1. Vaya a **Registros**
2. Seleccione componente
3. Filtre por:
   - Fecha
   - Nivel de severidad
   - Palabra clave
4. Exporte si es necesario

### Configuración de registros

- Nivel de detalle
- Rotación de archivos
- Retención de registros
- Tamaño máximo

## Mantenimiento

### Tareas de mantenimiento

1. Limpiar base de datos
2. Optimizar índices
3. Verificar integridad
4. Actualizar estadísticas

### Diagnosticador

- Ejecutar prueba de sistema
- Verificar dependencias
- Revisar recursos
- Generar reporte

## Resolución de problemas

- Verificar espacio en disco
- Revisar conectividad de red
- Comprobar registros de errores
- Contactar con soporte técnico
