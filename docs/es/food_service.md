# Compras en Línea

>Introducción

![Compras](_images/18.png)

En el `Custom menu`, los administradores configuran las categorías de productos para la venta en línea y los detalles de los productos, además de ver y gestionar el estado de los pedidos.

## Categoría de Compras

>Introducción

![Categoría de Compras](_images/18-1.png)

En `Shopping Category`, el administrador crea, modifica y elimina categorías de artículos.

Presione el botón `Add` para crear una nueva categoría de artículo.

![Categoría de Compras](_images/18-2.png)

<font color="red">**ID**</font>: En `ID`, el sistema IPTV crea automáticamente un número de ID que se usa para ordenar las distintas categorías; cuanto menor sea el número, más alta será su posición en la lista.

<font color="red">**Name**</font>: El `Name` se usa para mostrar el título de la categoría.

## Compras

>Introducción

![Compras](_images/18-3.png)

En la página `Shopping`, el administrador mantiene los productos del comercio electrónico y gestiona productos específicos mediante operaciones de creación, edición y eliminación.

Presione el botón `Add` para crear un nuevo artículo para la tienda en línea.

![Compras](_images/18-4.png)

<font color="red">**Imagen**</font>: En `Imagen`, cargue imágenes del producto; las imágenes se mostrarán en el terminal. Actualmente solo se admite subir una imagen.

<font color="red">**Nombre**</font>: `Nombre` se usa para identificar el nombre del producto que se muestra en el terminal.

<font color="red">**Precio**</font>: En `Precio`, se muestra el precio unitario al que se vende el producto.

<font color="red">**Categoría de Compras**</font>: Seleccione la `Categoría de Compras` para el producto; el producto aparecerá en la lista de la categoría correspondiente en el terminal.

<font color="red">**Descripción**</font>: En `Descripción` se escriben la información específica y los detalles sobre este producto.

## Estado de Pedidos

>Introducción

![Pedidos](_images/18-5.png ':size=40%') ![Pedidos](_images/18-6.png ':size=40%')

En esta página el administrador puede ver los detalles del pedido enviado por el suscriptor desde el terminal. Cuando el administrador recibe el pedido del suscriptor, puede `recibir` el pedido usando el botón RECEIVE o `cancelar` el pedido del suscriptor usando el botón DELETE. Tras aceptar el pedido del suscriptor, cuando el pedido se complete, el administrador puede hacer clic en el botón Finish para `completar` todo el proceso del pedido.
