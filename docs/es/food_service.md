# Configuración de Alimentos

>Introducción

![Alimentos](_images/food/food_1.png) 

En el `Servicio del Hotel`, los administradores configuran los tipos de alimentos y los elementos del menú para el sistema de pedidos IPTV en la habitación, y pueden ver, aceptar, completar o cancelar los pedidos de los huéspedes en tiempo real. Los huéspedes realizan pedidos directamente desde su televisor, y los administradores gestionan todo el flujo de trabajo de pedidos a través de la plataforma.

## Tipo de Alimento

>Introducción

![Tipo de Alimento](_images/food/food_2.png) 

En `Tipo de Alimento`, el administrador crea, modifica y elimina categorías de artículos.

Presione el botón `Agregar` para crear una nueva categoría de artículo

![Tipo de Alimento](_images/food/food_3.png)

<font color="red">**ID**</font>: En `ID`, el sistema IPTV crea automáticamente un número de ID que se utiliza para ordenar las diferentes categorías; cuanto menor sea el número, más avanzada será su posición en la lista.

<font color="red">**Nombre**</font>: El `Nombre` se usa para mostrar el alimento de la categoría.

## Lista de Alimentos

>Introducción

![Lista de Alimentos](_images/food/food_4.png)

En la página `Lista de Alimentos`, el administrador gestiona los elementos de alimentos disponibles para pedidos en habitaciones mediante la creación, edición o eliminación de productos según sea necesario.

Presione el botón `Agregar` para crear un nuevo elemento para pedidos en línea.

![Lista de Alimentos](_images/food/food_5.png)

<font color="red">**Imagen**</font>: En `Imagen`, cargue imágenes de productos y las imágenes se mostrarán en el terminal; actualmente solo se admite la carga de una imagen.

<font color="red">**Imagen de Fondo**</font>: La imagen de fondo que se mostrará en el terminal cuando el usuario seleccione o haga clic en el elemento de alimento.

<font color="red">**Nombre**</font>: `Nombre` se usa para identificar el nombre del alimento que se muestra en el terminal.

<font color="red">**Precio**</font>: En `Precio`, se usa para mostrar el precio unitario al que se vende el alimento.

<font color="red">**Tipo de Alimento**</font>: Seleccione el `Tipo de Alimento` para el alimento, y el alimento aparecerá en la lista de categorías correspondiente en el terminal.

<font color="red">**Descripción**</font>: En `Descripción` se usa para describir información específica y detalles sobre este alimento.

## Estado de Pedidos

>Introducción

![Estado de Pedidos](_images/food/food_6.png ':size=40%') 

En esta página, el administrador puede ver los detalles del pedido de artículos enviado por el suscriptor desde el terminal. Cuando el administrador recibe el pedido del suscriptor, puede `recibir` el pedido usando el botón RECEIVE o `cancelar` el pedido del suscriptor usando el botón DELETE. Después de aceptar el pedido del suscriptor, cuando el pedido se complete, el administrador puede hacer clic en el botón Finish nuevamente para `completar` todo el proceso del pedido.
