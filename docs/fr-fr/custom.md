# Paramètres personnalisés

> Introduction

![Custom Setting](_images/custom/custom_1.png)

Dans le menu `Custom`, l'administrateur décide d'afficher ou non le message de bienvenue sur la page d'accueil, configure la commande de repas en ligne et les informations sur les points d'intérêt.

## Welcome Words

> Introduction

![Welcome Word](_images/custom/custom_2.png)

Configurer ici le message de bienvenue affiché sur le terminal lors du check-in.

Cliquez sur `Add` pour créer un message.

![Welcome Word - Add](_images/custom/custom_3.png)

<font color="red">**ID**</font> : généré par le système.

<font color="red">**Clients**</font> : sélectionner les terminaux qui afficheront ce message.

<font color="red">**Welcome Words**</font> : saisir le texte du message de bienvenue.

## Food Type

> Introduction

![Food Type](_images/custom/custom_4.png)

Créer des catégories d'aliments pour distinguer les menus.

Cliquez sur `Add` pour ajouter une catégorie.

![Food Type - Add](_images/custom/custom_5.png)

<font color="red">**ID**</font> : généré par le système.

<font color="red">**Name**</font> : nom de la catégorie.

## Food

> Introduction

![Food](_images/custom/custom_6.png)

Liste des plats réservables (image, nom, prix, catégorie, description, nombre de réservations). Ajout, édition, suppression possibles.

Cliquez sur `Add` pour ajouter un plat.

![Food - Add](_images/custom/custom_7.png)

<font color="red">**Image**</font> : téléverser l'image (PNG/JPG) du plat.

<font color="red">**Name**</font> : nom affiché.

<font color="red">**Price($)**</font> : prix du plat.

<font color="red">**Food Type**</font> : catégorie du plat.

<font color="red">**Description**</font> : description visible côté client.

## Scenery

> Introduction

![Scenery](_images/custom/custom_8.png)

Ajouter des images et descriptions pour présenter les installations, services ou attractions proches.

Cliquez sur `Add` pour créer une fiche.

![Scenery - Add](_images/custom/custom_9.png)

Téléversez l'image d'aperçu et le titre, puis soumettez. Ajoutez ensuite des images détaillées et du texte via l'icône plus.

![Scenery - Add2](_images/custom/custom_10.png ':size=40%') ![Scenery - Add3](_images/custom/custom_11.png ':size=40%')
