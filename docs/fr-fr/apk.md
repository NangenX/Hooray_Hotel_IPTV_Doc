Dans le menu `Setting` de l'APK, vous configurez l'adresse de communication entre le terminal et le serveur IPTV ainsi que les autres paramètres de l'APK.

## Network Setting

> Introduction

Dans `Network Setting`, vous consultez et définissez l'adresse IP et le port du serveur IPTV, ainsi que l'IP locale et l'adresse MAC du terminal.

![Network Setting ](_images/apk/apk_2.png)

<font color="red">**Server IP**</font> : saisissez l'IP ou le nom de domaine pointant vers le serveur IPTV. Si l'accès échoue, reconfigurez pour qu'il soit joignable.

<font color="red">**Server Port**</font> : renseignez le port du serveur (identique à celui de l'interface Web) pour établir la communication.

<font color="red">**Local Network**</font> : IP locale obtenue par l'appareil (Android). Vérifiez qu'elle peut communiquer avec l'IP/nom de domaine du serveur.

<font color="red">**Local MAC**</font> : adresse MAC du terminal, utilisée comme identifiant unique pour la gestion et l'association au numéro de chambre.
