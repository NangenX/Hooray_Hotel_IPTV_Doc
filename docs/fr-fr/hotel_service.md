# Services hôteliers

> Introduction

![Food](_images/hotel/hotel_1.png)

Le module `Hotel Service` permet de configurer les catégories et services proposés via l'IPTV en chambre. Les administrateurs peuvent voir, accepter, terminer ou annuler les commandes en temps réel. Les invités commandent depuis la TV, l'admin gère tout le flux.

## Hotel Service Type

> Introduction

![Hotel Service Type](_images/hotel/hotel_2.png)

Créer, modifier, supprimer les catégories de services hôteliers.

![Hotel Service Type](_images/hotel/hotel_3.png)

<font color="red">**ID**</font> : généré automatiquement (plus petit = ordre d'affichage plus haut).

<font color="red">**Name**</font> : nom de la catégorie de service.

## Hotel Service List

> Introduction

![Hotel Service](_images/hotel/hotel_5.png)

![Hotel Service](_images/hotel/hotel_4.png)

Gérer les services commandables en chambre (ajout, édition, suppression).

![Hotel Service List](_images/hotel/hotel_6.png)

<font color="red">**Image**</font> : image du service (1 image) affichée sur le terminal.

<font color="red">**Backgroud picture**</font> : image d'arrière-plan quand l'utilisateur sélectionne le service.

<font color="red">**Name**</font> : nom affiché du service.

<font color="red">**Price**</font> : prix du service.

<font color="red">**Hotel Service Type**</font> : catégorie liée (affichage dans la liste correspondante).

<font color="red">**Description**</font> : détails du service.

## Order Status

> Introduction

![Order Status](_images/hotel/hotel_7.png ':size=40%')

Vue des commandes envoyées depuis les terminaux. Actions : `receive` pour accepter, `delete` pour annuler, puis `finish` pour clôturer une fois le service rendu.
