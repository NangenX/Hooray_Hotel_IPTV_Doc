# Paramétrage météo

> Introduction

![Weather Setting](_images/weather/weather_1.png)

Dans `Weather`, l'administrateur ajoute/supprime/modifie les villes cibles. Le système récupère automatiquement la météo à 7 jours via une interface tierce puis la transmet aux terminaux pour affichage.

![Weather Setting](_images/weather/weather_2.png)

Cliquez sur `Add` pour ajouter une ville.

![Weather Setting](_images/weather/weather_3.png)

<font color="red">**ID**</font> : identifiant unique (plus petit = plus haut dans la liste).

<font color="red">**Country**</font> : pays de la ville.

<font color="red">**City**</font> : choisir la ville cible dans la liste déroulante.
