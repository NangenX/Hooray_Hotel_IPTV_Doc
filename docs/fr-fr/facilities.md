# Paramètres des installations

> Introduction

![Facilities Setting](_images/facilities/facilities_1.png)

Ici, l'administrateur gère l'affichage du message d'accueil, la commande de repas et les informations sur les lieux à proximité.

## Welcome Words

> Introduction

![Welcome Word](_images/custom/custom_2.png)

Configurer le message de bienvenue affiché lors du check-in sur le terminal.

![Welcome Word - Add](_images/custom/custom_3.png)

<font color="red">**ID**</font> : généré automatiquement.

<font color="red">**Clients**</font> : terminaux qui afficheront le message.

<font color="red">**Welcome Words**</font> : texte du message de bienvenue.

## Food Type

> Introduction

![Food Type](_images/custom/custom_4.png)

Créer les catégories d'aliments.

![Food Type - Add](_images/custom/custom_5.png)

<font color="red">**ID**</font> : généré automatiquement.

<font color="red">**Name**</font> : nom de la catégorie.

## Food

> Introduction

![Food](_images/custom/custom_6.png)

Liste des plats réservables avec image, nom, prix, catégorie, description. Ajout/édition/suppression possibles.

![Food - Add](_images/custom/custom_7.png)

<font color="red">**Image**</font> : image PNG/JPG du plat.

<font color="red">**Name**</font> : nom du plat.

<font color="red">**Price($)**</font> : prix du plat.

<font color="red">**Food Type**</font> : catégorie liée.

<font color="red">**Description**</font> : description affichée côté utilisateur.
