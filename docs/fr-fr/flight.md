# Paramétrage des vols

> Introduction

![Flight Info Setting](_images/flight/flight_1.png)

Dans le menu `Flight Info`, l'administrateur ajoute/modifie/supprime les aéroports des villes cibles. Le système récupère automatiquement les arrivées/départs via une interface tierce, puis transmet les informations aux terminaux qui les affichent.

![Flight Info Setting](_images/flight/flight_2.png)

Cliquez sur `Add` pour créer un aéroport.

![Flight Info Setting](_images/flight/flight_3.png)

<font color="red">**ID**</font> : identifiant (doit être unique). Plus il est petit, plus l'élément est haut dans la liste.

<font color="red">**Airport**</font> : nom de l'aéroport affiché sur le terminal.

<font color="red">**Country**</font> : pays de l'aéroport.

<font color="red">**City**</font> : ville de l'aéroport.

<font color="red">**Airport Code(IATA Code)**</font> : renseigner précisément le code IATA pour obtenir les infos de vol correctes.
