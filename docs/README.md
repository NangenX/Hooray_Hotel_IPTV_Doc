# Hooray IPTV Online Documentation

> Based on Hooray Hotel IPTV Platform v1.1.3_Release_Date_2025-12-11.

## Overview
This site documents how to configure and operate the Hooray Hotel IPTV management platform. It provides step-by-step guidance for installation, configuration, and daily operations of the system.

## Quick Feature Overview
- **Device & Client Management:** Register and manage set-top boxes and clients, group devices, push configurations and OTA updates via the management console.
- **Live TV:** Configure live channels, sources, EPG, and manage channel packages and scheduling.
- **Video On Demand (VOD):** Upload, categorize, and manage on-demand content libraries with playback and metadata controls.
- **Advertising & Promotions:** Schedule and manage in-room ads, channel ads, and promotional content with media management support.
- **Packages & Settings:** Create channel/package bundles, configure access rules, and manage package-related settings.
- **Hotel Services & Integrations:** Provide hotel-centric features such as facility info, flight information, weather, shopping, and guest messaging.
- **Permissions & Security:** Role-based access control, operator permissions, and authority management for secure operations.
- **System Administration:** System settings, backups, logging, and maintenance tools for stable operation and troubleshooting.
- **Localization & Assets:** Documentation and localized assets available under `docs/` and `docs/ru-ru/`, with images in `docs/_images/`.

## Getting Started
- Read `quickstart.md` for an initial setup walkthrough.
- Configure streaming and channel data using `live-setting.md` and `vod_setting.md`.
- Deploy and manage client APKs with `apk.md` and monitor devices via `client_manage.md`.
